# Number System

Number System is the foundation of all quantitative aptitude questions. It covers divisibility rules,
HCF/LCM, remainders (Cyclicity, Euler's theorem), unit digits, number of factors, and properties
of natural/prime/composite numbers.

- **TCS NQT Frequency**: 🔴 High (4–6 questions per slot)
- **Difficulty**: Medium to Hard
- **Weightage**: ~15% of Quant section
